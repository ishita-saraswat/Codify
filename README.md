# Pre-Requisites

1. Node is installed (IMPORTANT)
2. Nodemon is installed (optional)

# Instructions

1. Clone the Repository
2. cd .\client => npm i => npm start (Front-End)
2. cd .\server => npm i => npm run start (Back-End)
